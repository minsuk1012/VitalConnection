# 썸네일 레퍼런스 특징 분석

최근 수집된 여신티켓(K-Beauty 플랫폼) 등의 뷰티 시술 썸네일 레퍼런스 4종을 분석한 결과, 다음과 같은 공통적인 디자인 특징을 확인했습니다.

## 1. 모델 중심의 고품질 비주얼
- **피사체 강조**: 고품질의 프로페셔널한 모델 사진에 크게 의존합니다. 시술 부위(예: 다리, 얼굴)를 직관적으로 보여주거나, 깨끗하고 선명한 피부를 가진 모델의 측면/정면 샷을 전면에 내세웁니다.
- **여백 활용**: 텍스트가 들어갈 반대편 공간을 깔끔하게 비워두거나 모델을 하프톤(반쪽)으로 배치하여 시선이 분산되지 않게 유도합니다.

## 2. 대비가 강렬한 배경 처리
- **솔리드 컬러 기반**: 어두운 와인 레드, 파스텔 톤 블루, 밝은 옐로우 등 단색(solid) 배경을 사용해 피사체와 텍스트를 동시에 돋보이게 합니다.
- **분할 레이아웃(Split Layout)**: 사선이나 직선으로 영역을 나누어, 한쪽은 모델 컷, 다른 한쪽은 강렬한 보색/단색 배경을 깔아 정보 가독성을 극대화하는 방식이 자주 쓰입니다.

## 3. 강력하고 직관적인 타이포그래피 (Typography)
- **명확한 시술명**: 가장 핵심이 되는 시술명(예: '종알매끈주사', '여성 탈모케어', '점제거', '슈링크')은 가장 굵고 거대한 고딕 계열 폰트로 배치됩니다.
- **눈길을 끄는 가격 표시 (핵심!)**: 가격 정보는 썸네일 내에서 가장 눈에 띄는 요소입니다. 주로 'X.X만', 'X만X천원'과 같이 소수점이나 만원 단위를 크게 표시하며, 핑크, 오렌지 둥 강렬한 포인트 컬러나 아웃라인 이펙트를 적용하여 돋보이게 만듭니다.
- **감성적인 서브 카피**: 핵심 타이틀 근처에 '매끈한 다리의 완성은..', '자연스럽게 환해지는 얼굴'과 같은 서브 카피가 손글씨(핸드라이팅) 폰트나 얇은 폰트로 배치되어 차분하게 혜택(Benefit)을 전달합니다.

## 4. 정보의 계층 구조 (Hierarchy)
- **로고**: 좌측 상단이나 우측 상단 모서리에 작게 병원/플랫폼 로고가 위치.
- **시선 흐름 (Z-pattern / F-pattern)**: 모델(이미지) → 메인 카피(시술명) → 강렬한 가격 순으로 눈길이 이어지며, 0.5초 안에 해당 시술이 얼마인지 파악할 수 있도록 구성되어 있습니다.

---
**요약:**
성공적인 뷰티 시술 썸네일은 **①신뢰감을 주는 모델 비주얼 + ②가독성 높은 굵은 고딕 텍스트 + ③극대화된 가격 포인트 컬러** 공식으로 요약될 수 있습니다. 복잡한 효과보다는 극단적인 대비와 타이포그래피의 강약 조절로 승부합니다.
