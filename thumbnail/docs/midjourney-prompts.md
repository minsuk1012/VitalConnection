## 파라미터 설명

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| `--ar 1:1` | 정방형 | 썸네일 1080×1080 기준 |
| `--style raw` | 사실적 | AI 과보정 최소화 |
| `--v 6.1` | 최신 버전 | 품질 최상 |
| `--stylize 50` | 낮은 스타일화 | 자연스러운 인물 |
| `--stylize 0` | **누끼용** | 배경 최대한 단색 flat하게 |

> 누끼용은 `--stylize 0` 사용 — 배경에 그라디언트/텍스처 생성 억제
